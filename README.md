# 📈 Stock Watchlist App

A sleek, minimal stock watchlist web app built with **Vite + React + TypeScript**, styled with **TailwindCSS**, and powered by the **Twelve Data API**. Easily search, track, and monitor your favorite stocks in real-time.

---

## 🚀 Features

- 🔍 Search for any stock by symbol (e.g. `AAPL`, `MSFT`)
- 🧾 Live stock quotes with real-time prices and percent changes
- 🗂 Add and remove stocks from your personal watchlist
- 🎨 Clean, minimalist dark UI
- 🧠 Built with modern React, hooks, and TypeScript
- ⚡️ Super fast thanks to Vite and lightweight components

---

## 🛠️ Tech Stack

- **Frontend:** React + TypeScript + Vite
- **Styling:** Tailwind CSS
- **API:** [Twelve Data](https://twelvedata.com/) (real-time stock quotes)
- **Deployment:** Vercel (or any static hosting)
